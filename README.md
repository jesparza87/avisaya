# AvisaYa

**AvisaYa** — Pager digital para hostelería. QR + push notification, sin app.

## Stack

- **Backend**: Node.js + Express + TypeScript + PostgreSQL + Drizzle ORM
- **Frontend**: React 18 + Vite + TypeScript + Tailwind CSS
- **Auth**: JWT con httpOnly cookies

## Requisitos

- Node.js 20+
- PostgreSQL 14+
